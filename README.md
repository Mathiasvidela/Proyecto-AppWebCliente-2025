Descripción del proyecto

Esta es una tienda online responsiva desarrollada completamente con HTML, CSS y JavaScript vanilla, sin frameworks.
El sitio muestra un catálogo dinámico de productos dividido por categorías (Celulares, Smart Watch, Headphones), permite ver detalles individuales y gestionar un carrito de compras funcional.

Los productos y toda la información del catálogo se obtienen mediante Airtable, consumiendo su API REST directamente desde JavaScript.
El sitio también incluye un panel de administración CRUD, donde se pueden crear, editar y eliminar productos, todo conectado al backend de Airtable.

- Además, cuenta con funcionalidades clave como:

- Navegación dinámica por categorías mediante query params.

- Carga de productos con fetch desde la API.

- Carrito con persistencia usando localStorage.

- Validaciones, toasts personalizados y contadores de stock.

- Interfaces completamente maquetadas y estilizadas con CSS puro, sin Bootstrap ni librerías externas.
